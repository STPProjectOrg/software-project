Features

Umsetzung in Django
Dieses Projekt wurde in der Programmiersprache Python auf Basis des Django-Frameworks erstellt. Dadurch ist ein Robustes Framework gewährleistet, welches umfangreiche Anpassungen ermöglicht, von dem Nutzer vielseitig profitieren!
Sicherheit durch Django Admin
Um Nutzerdaten zu schützen wurde als Grundlage für die Authentifizierung auf Django-Admin zurückgegriffen. Hierdurch konnte auf Komponenten für die Nutzerverwaltung zurückgegriffen werden, die sich bereits im großen Maßstab bewährt haben. Aufbauen hierrauf wurde der Login und die Registrierung angepasst. Für den Fall, dass Nutzer ihr Passwort vergessen wurde sogar das Zurücksetzen über Email hinzugefügt.
Umfangreiche Kursdaten
Für die Bereitstellung der Kurse greifen wir auf tagesaktuelle Kurse von Cryptocompare.com zurück, welche über einen Background-Job täglich über eine API-Schnittstelle abgerufen und in unsere Datenbank gespeichert werden. Für die bessere Darstellung wurden die Daten der vergangenen Jahre für eine Vielzahl von Kryptowährungen gespeichert.
Detailliertes Portfolio
Für jede Kryptowährung wird automatisch eine Detailseite erstellt. Von hier aus können Werte auf die Merkliste gesetzt werden oder dem Portfolio hinzugefügt werden. Aus den hinzugefügten Werten wird ein taggenauer Kursverlauf erstellt und eine Vielzahl von KPI angezeigt.
Attraktive UI
Das UI baut auf Bootstrap v5.3 und bietet so eine menge an herausragenden Komponenten, welche bei der Gestaltung umfangreich zum Einsatz kamen. Darüber hinaus wurde das bestehende Design um über 300 Zeilen SCCS erweitert, um Schriftart, Farben und vieles mehr zu individualisieren. Es wurde ein neuer Name überlegt und mit AI ein Logo erstellt.
Interaktive Charts
Wir verwenden Chart.js um Kursdiagramme zu erstellen, mit denen unsere Nutzer interagieren können. Natürlich wurde auch diese Diagramme grafisch grundlegend überarbeitet und funktionstechnisch um zum Beispiel eine Filterfunktion erweitert.
Individualisierbares Profil
Das Nutzerprofil lässt sich durch ein Profil-Bild und Banner individualisieren. Auch eine Biografie kann festgelegt werden.
Moderne Community-Funktionen
Wir bieten die Möglichkeit an anderen Nutzern zu folgen und auch selbst gefolgt zu werden. Außerdem können Posts erstellt werden, um seine Ansichten und Meinungen mit der Community zu teilen.
