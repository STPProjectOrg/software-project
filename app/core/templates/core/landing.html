{% trans "Dieses Projekt wurde in der Programmiersprache Python auf Basis des Django-Frameworks erstellt. Dadurch ist ein Robustes Framework gewährleistet, welches umfangreiche Anpassungen ermöglicht, von dem Nutzer vielseitig profitieren!" %}
{% trans "Sicherheit durch Django Admin" %}
{% trans "Um Nutzerdaten zu schützen wurde als Grundlage für die Authentifizierung auf Django-Admin zurückgegriffen. Hierdurch konnte auf Komponenten für die Nutzerverwaltung zurückgegriffen werden, die sich bereits im großen Maßstab bewährt haben. Aufbauen hierrauf wurde der Login und die Registrierung angepasst. Für den Fall, dass Nutzer ihr Passwort vergessen wurde sogar das Zurücksetzen über Email hinzugefügt." %}
{% trans "Umfangreiche Kursdaten" %}
{% trans "Für die Bereitstellung der Kurse greifen wir auf tagesaktuelle Kurse von Cryptocompare.com zurück, welche über einen Background-Job täglich über eine API-Schnittstelle abgerufen und in unsere Datenbank gespeichert werden. Für die bessere Darstellung wurden die Daten der vergangenen Jahre für eine Vielzahl von Kryptowährungen gespeichert." %}
{% trans "Detailliertes Portfolio" %}
{% trans "Für jede Kryptowährung wird automatisch eine Detailseite erstellt. Von hier aus können Werte auf die Merkliste gesetzt werden oder dem Portfolio hinzugefügt werden. Aus den hinzugefügten Werten wird ein taggenauer Kursverlauf erstellt und eine Vielzahl von KPI angezeigt." %}
